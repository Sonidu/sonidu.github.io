/* La posición del scrip determina como y cuando se carga*/

La carga de la web es secuelcial, de arriba a abajo, si ponemos un script arriba y tarda mucho, la pagina web no cargará hasta que el script esté cargado, por eso conviene pensar donde colocar el script, salvo que se usar "defer" que hará que el script se cargue de forma simultanea al contenido del script.

Usaremos LET (let) para definir variables, por ejemplo let nombre = "Juan". Antiguamete se usaba var, pero da errores por lo que conviene cambiar a let.
